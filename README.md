# ProgramAi-Front

Projeto Interdisciplinar 2025, IFBA Campus Eunápolis

---

## 1 — Pré-Requisitos

* **Node.js:** recomendamos **Node 20.x (ex.: 20.19.0)**
* **Angular CLI:** `@angular/cli@latest`

O projeto foi desenvolvido na versão do Angular 20

---

## 2 — Instalação

### 2.1 — Clone o repositório

   ```bash
   https://github.com/Kaike-Marcelo/ProgramAi-Front.git
   ```
   ```bash
   cd ProgramAi-Front
   ```

---

### 2.2 — Instalar Angular CLI globalmente (caso ainda não tenha)

```powershell
npm install -g @angular/cli@latest
ng version
```

---

### 2.3 — Instalar dependências do frontend (versões recomendadas)

Execute o seguinte comando para instalar todas as dependências necessárias do projeto:

```bash
  npm install
```

---

## Executando o Projeto

Para iniciar o servidor de desenvolvimento e executar o projeto, use o comando:

```bash
ng serve
```
ou caso queira iniciar o aplicativo automaticamente em seu navegador padrão:
```bash
ng s -o
```
